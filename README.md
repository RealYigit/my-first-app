# AI Test App

A modern React to-do app built with Vite.

## Features

- Add tasks
- Delete tasks
- Save tasks in `localStorage`
- Light and dark theme support
- Responsive layout

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Usage

Type a task into the input, press **Add**, and manage items with **Delete**.
